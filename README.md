## Description

Test Project based on NestJS with typescript aimed to measure customer experience rates

## Technologies and Libraries
- Database - SQLlite
- REST API - NestJS
- Typescript
- TypeORM
- Swagger

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn start

# watch mode
$ yarn start:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e

# test coverage
$ yarn test:cov
```
